System Prerequisites

Complete setup guide for Windows to prepare your system for the Computer Networks laboratory

Important: Follow the Correct Order

Some installations require system restarts. Follow the recommended order below to avoid configuration issues.

1

Enable WSL2

Requires Restart

Windows Subsystem for Linux 2 provides a Linux kernel running on Windows. This is required for Docker Desktop to function properly.

  • Open PowerShell as Administrator (right-click ‚Üí Run as administrator)
  • Run the installation command shown below
  • Restart your computer when prompted
PowerShell (Administrator) 
# Install WSL2 with default Linux distribution
wsl --install

# After restart, verify installation
wsl --status
Verification Checklist
  • WSL2 is listed as the default version
  • A Linux distribution is installed (usually Ubuntu)
2

Install Docker Desktop

Requires Restart

Docker Desktop provides the containerization platform needed to run the laboratory environment consistently across all systems.

  • Download from docker.com/products/docker-desktop
  • Run the installer
  • Important: Check "Use WSL 2 instead of Hyper-V" during installation
  • Complete installation and restart your computer
  • After restart, launch Docker Desktop and wait for it to start (icon turns green)
Verification Checklist
  • Docker Desktop icon in system tray is green
  • Running docker --version shows version info
  • Running docker run hello-world succeeds
3

Install Wireshark

Required

Wireshark is the world's most popular network protocol analyzer. You'll use it to capture and analyze network traffic.

  • Download from wireshark.org/download.html
  • Select "Windows x64 Installer"
  • Run the installer with default options
  • Important: When prompted, install Npcap (required for packet capture)
  • During Npcap installation, check "Support raw 802.11 traffic"
Verification Checklist
  • Wireshark launches from Start menu
  • Network interfaces are visible in Wireshark
  • You can start a capture on your active interface
4

Python Packages

Required

Install the Python packages required for the lab scripts. Make sure Python 3.11+ is already installed on your system.

PowerShell / Command Prompt 
# Required package for Docker control
pip install docker

# Verify installation
pip show docker
Verification
  • pip show docker displays package information
  • python -c "import docker; print('OK')" prints OK
5

Optional Python Packages

Recommended

These packages enhance your network analysis capabilities and are recommended for advanced exercises.

PowerShell / Command Prompt 
# Scapy - powerful packet manipulation library
pip install scapy

# dpkt - fast packet parsing library
pip install dpkt

# Or install all at once
pip install scapy dpkt
