## Usage Notes

### For Instructors

1. **Slides:** Use ASCII diagrams directly in terminal-based presentations
2. **Handouts:** Convert to PNG using tools like `carbon.now.sh` or diagram editors
3. **Live drawing:** Recreate on whiteboard while explaining

### For Students

1. **Study aid:** Redraw diagrams from memory to test understanding
2. **Notes:** Copy ASCII versions into your notes
3. **Modification:** Adapt diagrams for your own scenarios
