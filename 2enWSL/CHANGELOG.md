# Changelog

All notable changes to the Week 2 WSL Starter Kit will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-06

### Added

- Initial release of Week 2 WSL Starter Kit
- Concurrent TCP server implementation with threading support
- UDP server with custom application protocol (ping, upper, lower, reverse, time)
- Docker Compose configuration for isolated lab environment
- Python-based setup and verification scripts
- Comprehensive test suite with smoke tests
- Traffic capture utilities for Wireshark integration
- Detailed documentation following Anderson-Bloom taxonomy objectives
- Homework exercises with solution stubs
- Troubleshooting guide for common issues

### Features

- **TCP Exercise (ex_2_01_tcp.py)**: Multi-mode server (threaded/iterative), client, and load testing
- **UDP Exercise (ex_2_02_udp.py)**: Interactive and single-command client modes
- **Docker Environment**: Pre-configured container with all necessary tools
- **Capture Scripts**: Integration with tcpdump and Wireshark
- **Demo Scripts**: Automated demonstrations for instructor use

### Documentation

- README.md with complete laboratory guide
- Theory summary covering OSI and TCP/IP models
- Commands cheatsheet for quick reference
- Troubleshooting guide with common issues and solutions
