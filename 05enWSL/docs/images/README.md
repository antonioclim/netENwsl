## Viewing Diagrams

**SVG files:** Open directly in any web browser or VS Code.

**Mermaid files:** Use one of:
- VS Code with Mermaid extension
- [Mermaid Live Editor](https://mermaid.live/)
- GitHub renders `.mermaid` files automatically
