üéØ Why struct?

Network protocols use binary data with fixed-size fields. The struct module converts between bytes and Python types.

üì¶ IP Header (simplified)

Bytes 0-3: Version, IHL, TOS, Length
Bytes 4-7: ID, Flags, Fragment
Bytes 8-11: TTL, Protocol, Checksum
Bytes 12-15: Source IP
Bytes 16-19: Dest IP

üîß What struct does

pack() ‚Äî Python ‚Üí bytes

unpack() ‚Äî bytes ‚Üí Python

Specify exact format (size, endianness)

üîë Byte Order (Endianness)

! or > = Network byte order (Big Endian) ‚Äî ALWAYS use for network!

< = Little Endian (Intel native)

üìñ Parsing an IP Header

üêç parse_ip_header.py
import struct
import socket

# Raw IP header (20 bytes minimum)
ip_header = b'\x45\x00\x00\x3c\x1c\x46\x40\x00\x40\x06' + \
            b'\xb1\xe6\xc0\xa8\x01\x01\xc0\xa8\x01\x02'

# Unpacking (! = network byte order)
fields = struct.unpack('!BBHHHBBH4s4s', ip_header)

version_ihl = fields[0]
version = version_ihl >> 4        # First 4 bits
ihl = version_ihl & 0x0F          # Last 4 bits
ttl = fields[5]
protocol = fields[6]
src_ip = socket.inet_ntoa(fields[8])
dst_ip = socket.inet_ntoa(fields[9])

print(f"Version: IPv{version}")
print(f"TTL: {ttl}, Protocol: {protocol}")
print(f"{src_ip} ‚Üí {dst_ip}")

‚úçÔ∏è Building a Packet

üêç build_udp_packet.py
import struct

# Build UDP header
src_port = 12345
dst_port = 53      # DNS
payload = b"Hello DNS"
length = 8 + len(payload)  # Header (8) + data
checksum = 0       # Optional in UDP

# Pack: H=port(2), H=port(2), H=len(2), H=checksum(2)
udp_header = struct.pack('!HHHH', 
    src_port, dst_port, length, checksum
)

packet = udp_header + payload
print(f"UDP Header: {udp_header.hex()}")
print(f"Packet: {len(packet)} bytes")

üìè struct Utilities

üêç struct_utils.py
import struct

# calcsize() - size in bytes
ip_format = '!BBHHHBBH4s4s'
print(f"IP header: {struct.calcsize(ip_format)} bytes")  # 20

udp_format = '!HHHH'
print(f"UDP header: {struct.calcsize(udp_format)} bytes")  # 8

# unpack_from() - unpack from an offset
packet = b'\x00' * 20 + struct.pack('!HH', 80, 443)
src, dst = struct.unpack_from('!HH', packet, offset=20)
print(f"Ports: {src} ‚Üí {dst}")
