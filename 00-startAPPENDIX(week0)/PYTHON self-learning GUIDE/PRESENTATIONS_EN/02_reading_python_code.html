class PingResult:
    """Stores the result of a ping operation."""
    host: str                          # Target hostname
    packets_sent: int = 4              # Default value
    packets_received: int = 0
    latency_ms: Optional[float] = None  # Can be None if failed
    
    def packet_loss_percent(self) -> float:
        """Calculate packet loss percentage."""
        if self.packets_sent == 0:
            return 100.0
        lost = self.packets_sent - self.packets_received
        return (lost / self.packets_sent) * 100

üß†

What does @dataclass generate?

ADatabase connections
B__init__, __repr__, comparison methods
CNetwork packets
DTest functions
Hint 1/3

Dataclasses reduce boilerplate code for data-holding classes.

Hint 2/3

They auto-generate constructor and string representation.

Hint 3/3

Answer B: __init__ (constructor), __repr__ (string), and ==, <, > methods.

üß†

What does Optional[float] mean?

AThe value is optional in function calls
BThe value can be float OR None
CThe value is read-only
DThe value must be positive
Hint 1/3

Think about what happens when a ping fails - there's no latency value.

Hint 2/3

Optional[X] is equivalent to Union[X, None].

Hint 3/3

The value can be a float (e.g., 15.5) or None if unavailable.
