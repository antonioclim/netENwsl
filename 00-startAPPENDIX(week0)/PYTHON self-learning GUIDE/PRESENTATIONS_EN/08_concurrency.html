ü§î The Problem: One Client Blocks All

‚ùå Sequential Server

while True:
    conn, addr = server.accept()
    # Client 1 connects
    handle_client(conn)  # BLOCKED!
    # Client 2 waits...
    # Client 3 waits...

Each client must wait for the previous one to finish!

‚úÖ Concurrent Server

while True:
    conn, addr = server.accept()
    # New thread for each client!
    Thread(target=handle, args=(conn,)).start()
    # Immediately accept next

All clients are served in parallel!

üßµ Threading Basics

üêç threading_basics.py
from threading import Thread
import time

def task(name: str, duration: int):
    print(f"üöÄ {name} starting")
    time.sleep(duration)
    print(f"‚úÖ {name} done after {duration}s")

# Create threads
t1 = Thread(target=task, args=("Client A", 2))
t2 = Thread(target=task, args=("Client B", 1))

# Start them (run in parallel!)
t1.start()
t2.start()

# Wait for both to finish
t1.join()
t2.join()
print("üèÅ All threads finished")

üñ•Ô∏è Threaded TCP Server

üêç server_threaded.py
import socket
from threading import Thread

def handle_client(conn, addr):
    print(f"üîó {addr} connected")
    try:
        while True:
            data = conn.recv(1024)
            if not data: break
            conn.sendall(data.upper())
    finally:
        conn.close()

with socket.socket() as server:
    server.setsockopt(socket.SOL_SOCKET, socket.SO_REUSEADDR, 1)
    server.bind(("0.0.0.0", 9090))
    server.listen()
    
    while True:
        conn, addr = server.accept()
        Thread(target=handle_client, args=(conn, addr), daemon=True).start()

üèä ThreadPoolExecutor

Instead of creating threads manually, use a pool with a fixed number of workers:

üêç server_pool.py
from concurrent.futures import ThreadPoolExecutor

with socket.socket() as server:
    server.bind(("0.0.0.0", 9090))
    server.listen()
    
    with ThreadPoolExecutor(max_workers=10) as pool:
        while True:
            conn, addr = server.accept()
            pool.submit(handle_client, conn, addr)

üí° Pool Advantages

‚úì Limits thread count ‚úì Reuses threads ‚úì Avoids creation overhead

‚ùå Race Condition

counter = 0
def increment():
    global counter
    for _ in range(100000):
        counter += 1  # NOT ATOMIC!

‚úÖ With Lock

from threading import Lock
lock = Lock()
def increment():
    global counter
    for _ in range(100000):
        with lock:
            counter += 1

üëª Daemon Threads

Normal Thread

Program waits for ALL normal threads to finish before exiting.

daemon=False (default)

Daemon Thread

Stops automatically when main thread ends. Perfect for servers!

daemon=True

# Client handlers are daemon threads
Thread(target=handle_client, args=(conn, addr), daemon=True).start()
# When server stops, handlers stop too
