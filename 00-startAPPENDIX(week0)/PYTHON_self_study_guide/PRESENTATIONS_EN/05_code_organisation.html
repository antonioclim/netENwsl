üì¶ Functions in Python

üêç functions.py
# Simple function
def calculate_checksum(data: bytes) -> int:
    """
    Calculate checksum of a packet.
    
    Args:
        data: Packet data as bytes
    
    Returns:
        16-bit checksum
    """
    if len(data) % 2:
        data += b'\x00'
    
    total = 0
    for i in range(0, len(data), 2):
        word = (data[i] << 8) + data[i+1]
        total += word
    
    return ~total & 0xFFFF

# Function with optional parameters
def format_ip(bytes: list, separator: str = ".") -> str:
    """Converts list of bytes to IP string."""
    return separator.join(str(o) for o in bytes)

# Usage
ip = format_ip([192, 168, 1, 1])
print(ip)  # "192.168.1.1"

üîë The if __name__ == "__main__" Pattern

This pattern allows a file to be both an importable modulee and an executable script.

üêç utils/network.py
"""Network utility functions."""

def ip_to_int(ip: str) -> int:
    """Convert IP string to integer."""
    parts = [int(p) for p in ip.split('.')]
    return (parts[0] << 24) + (parts[1] << 16) + (parts[2] << 8) + parts[3]

def int_to_ip(num: int) -> str:
    """Convert integer to IP string."""
    return '.'.join(str((num >> i) & 0xFF) for i in [24, 16, 8, 0])

# This block runs ONLY when the file is executed directly
# Does NOT run when the file is imported
if __name__ == "__main__":
    # Quick test
    test_ip = "192.168.1.1"
    num = ip_to_int(test_ip)
    print(f"{test_ip} ‚Üí {num} ‚Üí {int_to_ip(num)}")

üí° Why is it useful?

You can test the modulee by running it directly (python utils/network.py) but also import functions in other files without the tests running.

üìù Quiz

üß†

What does if __name__ == "__main__" do?

AMakes the file executable on Linux
BExecutes code only when the file is run directly, not when imported
CDefines the mandatory main function
DCreates an entry point for debugging
Hint 1/3

__name__ is a special variable. When you run a file directly, its value is "__main__".

Hint 2/3

When you import a modulee, __name__ becomes the modulee name, not "__main__".

Hint 3/3

Answer B: allows you to have test code that doesn't run when the modulee is imported.
