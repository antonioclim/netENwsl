üîå Ce is un Socket?

A socket is un endpoint of comunicare bidirectionala. Gandeste-te at el ca at o "priza" through which the application ta se connects at network.

Socket = (IP, Port, Protocol)

üìç Componentele unui Socket

Adres IP: Identific masina in network

Port: Identific the application on masin (0-65535)

Protocol: TCP or UDP

üéØ Porturi Uzuof

80 - HTTP, 443 - HTTPS

22 - SSH, 53 - DNS

8080, 9090 - Development

üì¶ TCP vs UDP ‚Äî Alegerea Protocolului

üîµ TCP (Transmission Control Protocol)

Characteristics:

  • Connection-oriented (requires handshake)
  • Garanteaz livrarea in ordine
  • Retransmite packageele pierdute
  • Flow control and congestion control

Folosit pentru: HTTP, SSH, Email, FTP

Socket type: SOCK_STREAM

üü† UDP (User Datagram Protocol)

Characteristics:

  • Connectionless (fire and forget)
  • Does not guarantee deliverya
  • Nu garanteaz ordinea
  • Overhead minim, latency small

Folosit pentru: DNS, Gaming, Streaming, VoIP

Socket type: SOCK_DGRAM

üîë Regul Simpla

TCP: Cand ai nevoie of reliability (data which NU must pierdute)

UDP: Cand ai nevoie of vitez and poti tolera losses (streaming, gaming)

üíª TCP Server ‚Äî Cod Complet

üêç server_tcp.py (din 02roWSL/src/exercises/) 
import socket

# Constante
HOST = "127.0.0.1"   # Localhost
PORT = 9090          # Port non-privilegiat

# Context manager: socket se closes automat!
with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as server:
    # Allow reutilizarea addressesi (evit "Address already in use")
    server.setsockopt(socket.SOL_SOCKET, socket.SO_REUSEADDR, 1)
    
    # Pasul 1: Bind at address and port
    server.bind((HOST, PORT))
    
    # Pasul 2: Start listening (max 5 connections in asteptare)
    server.listen(5)
    print(f"üñ•Ô∏è  TCP Server ascult on {HOST}:{PORT}")
    
    # Pasul 3: Accept connections (loop infinita)
    while True:
        conn, addr = server.accept()  # Block aici
        with conn:
            print(f"‚úÖ Conexiune of at {addr}")
            
            # Receive data
            data = conn.recv(1024)
            if not data:
                break
            
            print(f"üì© Primit: {data.decode('utf-8')}")
            
            # Send response
            raspuns = f"Server a primit: {data.decode('utf-8')}"
            conn.sendall(raspuns.encode('utf-8'))

üü† UDP ‚Äî Connectionless

UDP not are conexiune ‚Äî trimiti datagrame direct, far handshake.

Server UDP

import socket

with socket.socket(socket.AF_INET, socket.SOCK_DGRAM) as s:
    s.bind(("127.0.0.1", 9091))
    print("üü† Server UDP on :9091")
    
    while True:
        # recvfrom returneaz and adresa
        data, addr = s.recvfrom(1024)
        print(f"De at {addr}: {data.decode()}")
        
        # Answer to client
        s.sendto(b"ACK", addr)

Client UDP

import socket

with socket.socket(socket.AF_INET, socket.SOCK_DGRAM) as s:
    # Nu exist connect() - sendsm direct!
    server = ("127.0.0.1", 9091)
    
    s.sendto(b"Hello UDP!", server)
    print("üì§ Trimis datagram")
    
    # Optional: primim response
    data, addr = s.recvfrom(1024)
    print(f"üì• Answer: {data}")

üîë Diferente Cheie TCP vs UDP

TCP:SOCK_STREAM, connect(), send()/recv()
UDP:SOCK_DGRAM, sendto(addr)/recvfrom()

üîí Context Managers ‚Äî Instructiunea with

Context managers garanteaz c resursele are eliberate correct, even if apare o exception.

‚ùå Far with (risc of leak)

# If apare o exception befhours of close(),
# socket-ul ramane deschis!

sock = socket.socket(...)
sock.connect((host, port))
sock.send(data)           # Exceptie aici?
response = sock.recv(1024)
sock.close()              # Nu se executa!

‚úÖ Cu with (secure)

# Socket-ul se closes AUTOMAT
# at iesirea from bloc, garantat!

with socket.socket(...) as sock:
    sock.connect((host, port))
    sock.send(data)
    response = sock.recv(1024)
# Closere automat aici

üí° Regula of Aur

MEREU uses with for sockets (si files)! Este echivaslowul Python for try-finally from Java or RAII from C++.

üêç Pattern: Echo Server (returneaz ce receives) 
def echo_server(host: str, port: int) -> None:
    """Server which returneaz exact ce receives."""
    with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
        s.setsockopt(socket.SOL_SOCKET, socket.SO_REUSEADDR, 1)
        s.bind((host, port))
        s.listen()
        
        while True:
            conn, addr = s.accept()
            with conn:
                while True:
                    data = conn.recv(1024)
                    if not data:
                        break
                    conn.sendall(data)  # Echo: sends inapoi
üêç Pattern: Client with Timeout 
def client_cu_timeout(host: str, port: int, timeout: float = 5.0):
    """Client which not blocks at infinit."""
    with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
        s.settimeout(timeout)  # Seteaz timeout
        try:
            s.connect((host, port))
            s.sendall(b"Hello")
            response = s.recv(1024)
            return response
        except socket.timeout:
            print("‚è±Ô∏è Timeout - the server not raspunde")
            return None

‚ö†Ô∏è Gestionarea Erorilor in Socket Programming

üêç error_handling.py 
import socket
import sys

def conectare_securea(host: str, port: int) -> socket.socket | None:
    """Incearc conectarea with gestionare complet a errorslor."""
    try:
        sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        sock.settimeout(10.0)
        sock.connect((host, port))
        return sock
        
    except socket.timeout:
        print(f"‚ùå Timeout: {host}:{port} not raspunde")
        
    except socket.gaierror as e:
        print(f"‚ùå DNS Error: Nu pot rezolva '{host}'")
        
    except ConnectionRefusedError:
        print(f"‚ùå Conexiune refuzata: Portul {port} inchis")
        
    except OSError as e:
        print(f"‚ùå Eroare network: {e}")
    
    return None

üî¥ Exceptii Comune

ConnectionRefusedError ‚Äî Port inchis or firewall

socket.timeout ‚Äî Server not raspunde in time

socket.gaierror ‚Äî DNS not can rezolva hostname

BrokenPipeError ‚Äî Conexiunea a fost intrerupta
