üè∑Ô∏è Dataclasses - Structuri of Date Elegante

Python uses @dataclass for a crea structuri similare with struct from C. Dectimetorul genereaz automat __init__, __repr__, and alte metode.

Python with @dataclass

üêç Python 
from dataclasses import dataclass
from typing import Optional

@dataclass
class RezultatPing:
    """Rezultatul unui ping."""
    secventa: int
    rtt_ms: Optional[float]
    reusit: bool
    message: str = ""  # valoare implicita

# Utilizare
r = RezultatPing(
    secventa=1,
    rtt_ms=0.42,
    reusit=True
)
print(r.secventa)  # 1
print(r)  # RezultatPing(secventa=1, ...)

Echivaslow in C

üìÑ C 
typedef struct {
    int secventa;
    float rtt_ms;     // not can fi NULL

// Utilizare
RezultatPing r;
r.secventa = 1;

üîë Concepte Cheie

message: str = "" = parameter with valoare implicit (nu must specificat at creare)

Quiz 1: Dataclasses

Questiona 1 from 3

What does dectimetorul @dataclass in Python?
A Transform clasa intr-o function
B Genereaz automat __init__, __repr__ and alte metode for clasa of data
C Check tipurile at runtime and arunc errors
D Converteste clasa in format JSON automat
Hint 1/3

Gandeste-te at ce scrii in mod normal when creezi o class. Ce metode must to implementezi manual for a putea crea instante and a le afisa?

Hint 2/3

In C, when folosesti struct, must to initializezi manual each camp. @dataclass elimin this necesitate in Python.

Hint 3/3

@dataclass = "data class" = class for data. Genereaz constructor (__init__), reprezentare text (__repr__), comparatii (__eq__) automat.

üß†

Quiz 2: Type Hints

Questiona 2 from 3

What does Optional[float] in contextul type hints?
A Valoarea is obligatoriu un float
B Parametrul is optional and can fi omis
C Valoarea can fi fie float, fie None
D Valoarea va fi convertit automat at float
Hint 1/3

In contextul networking, of ce ar fi rtt_ms (Round-Trip Time) optional? Ce se intampl when un package not ajunge?

Hint 2/3

"Optional" in type hints not mean c parameterl can fi omis. Se refer at tipul valorii. Optional[X] = Aion[X, None]

Hint 3/3

If ping-ul fails, not avem un RTT valid. In loc to folosim o valoare magic (-1), folosim None for a indica "far valoare".

from dataclasses import dataclass
from typing import Optional

@dataclass
class InfoPachet:
    """Informatii despre un package of network."""
    ip_sursa: str
    ip_dest: str
    protocol: str
    dimensiune: int
    ttl: Optional[int] = None

# Creaz un package TCP
package_tcp = InfoPachet(
    ip_sursa="192.168.1.100",
    ip_dest="8.8.8.8",
    protocol="TCP",
    dimensiune=1500,
    ttl=64
)

# Creaz un package UDP far TTL specificat
package_udp = InfoPachet(
    ip_sursa="192.168.1.100",
    ip_dest="1.1.1.1",
    protocol="UDP",
    dimensiune=512
)

print("Pachet TCP:")
print(f"  De la: {package_tcp.ip_sursa}")
print(f"  Catre: {package_tcp.ip_dest}")
print(f"  TTL: {package_tcp.ttl}")

print(f"\nPachet UDP (TTL neinitializat): {package_udp.ttl}")
print(f"\nReprezentare completa:\n{package_tcp}")
Output

  De la: 192.168.1.100

  TTL: 64

Reprezentare completa:
