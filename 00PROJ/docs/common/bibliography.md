## How to Cite

When referencing materials in your project reports, use IEEE citation style:

**Journal Article:**
```
[#] A. Author, B. Author and C. Author, "Article title," Journal Name, vol. X, no. Y, pp. XX–YY, Month Year. doi: 10.XXXX/XXXXX
```

**Conference Paper:**
```
[#] A. Author, "Paper title," in Proc. Conference Name, City, Country, Month Year, pp. XX–YY.
```

**Book:**
```
[#] A. Author, Book Title, Xth ed. City, Country: Publisher, Year.
```

**Online Resource:**
```
[#] Organisation, "Document title," [Online]. Available: https://url.example.com/
```
