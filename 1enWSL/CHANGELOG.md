# Changelog

All notable changes to this laboratory kit will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-06

### Added

- Initial release of WEEK1_WSLkit
- Complete Python-based laboratory management scripts
- Docker Compose configuration for isolated lab environment
- Five structured exercises covering:
  - Network interface inspection
  - Connectivity testing with ping
  - TCP communication with netcat
  - Traffic capture and analysis
  - PCAP statistical processing
- Automated demonstration scripts
- Environment verification utilities
- Comprehensive documentation suite
- Homework assignments with solution stubs
- Troubleshooting guide

### Infrastructure

- WSL2 + Docker Desktop target platform
- Python 3.11+ scripting throughout
- Portainer CE integration for container management
- Wireshark-compatible PCAP output
