## RFCs (Request for Comments)

Essential protocol specifications:

| RFC | Title | Topic |
|-----|-------|-------|
| RFC 791 | Internet Protocol | IPv4 specification |
| RFC 793 | Transmission Control Protocol | TCP specification |
| RFC 768 | User Datagram Protocol | UDP specification |
| RFC 826 | Address Resolution Protocol | ARP specification |
| RFC 792 | Internet Control Message Protocol | ICMP specification |
| RFC 1918 | Address Allocation for Private Internets | Private IP ranges |
| RFC 2616 | HTTP/1.1 | HTTP protocol (superseded by RFC 7230-7235) |

**How to read RFCs:**
- Visit: https://www.rfc-editor.org/
- Search by RFC number or keyword
- Start with the Abstract and Introduction sections

## Recommended Learning Path

### Week 1 Focus

1. Review TCP/IP model basics (Kurose Chapter 1)
2. Practice with Wireshark sample captures
3. Read Julia Evans' "How does the internet work?" zine
4. Complete the Beej's Guide introduction

### For Advanced Study

1. Read relevant RFCs for protocols of interest
2. Set up GNS3 for complex network simulation
3. Participate in networking CTF challenges
4. Contribute to open-source networking projects
